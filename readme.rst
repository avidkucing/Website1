Requirement
************

- Apache
- MySQL database
- PHP < 7

************
Installation
************

Clone pada directory server.
Buat database "hisamitsu" pada database MYSQL dan import hisamitsu.sql
Sesuaikan pengaturan pada application/config untuk pengaturan port database.
